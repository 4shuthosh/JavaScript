# JavaScript String Operations 
This JavaScript program that performs various string operations. The operations are designed to help you learn and understand different ways to manipulate and analyze strings using JavaScript.

## Features
- Find the length of each string
- Slice a portion of each string
- Extract a substring from each string
- Extract a substring using `substr()` method
- Replace characters in each string
- Replace all occurrences of a character in each string
- Convert each string to uppercase
- Convert each string to lowercase
- Concatenate all strings
